# Boston Harbor Plumbing — Website

This is a simple, fast static site (HTML/CSS/JS).

## Open it

- Double-click `index.html`, or
- Serve the folder with any static server.

### Quick local server (Python)

If you have Python installed:

```bash
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Customize

- Update the phone number: search for `(617) 555‑0123`
- Update the email: `service@bostonharborplumbing.com`
- Replace business name/logo text in `index.html`
